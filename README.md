# Books Scraper & Analysis Project

This project scrapes book data from Books to Scrape and analyzes prices and ratings using Python.

## Project Description
The project consists of two parts:
1. Web scraping part: collects book titles, prices, and ratings from the first 3 pages of Books to Scrape using requests and BeautifulSoup.
2. Data analysis part: calculates average, minimum, and maximum prices, and creates a histogram to visualize price distribution using pandas, matplotlib, and seaborn.

## How to Run
1. Install required libraries: pandas, beautifulsoup4, requests, matplotlib, seaborn.
2. Run the scraping code to collect data.
3. Run the analysis code to view statistics and the price distribution graph.
